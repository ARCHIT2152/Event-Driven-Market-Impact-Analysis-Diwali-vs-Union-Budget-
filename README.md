# Event-Driven-Market-Impact-Analysis-Diwali-vs-Union-Budget-
This project analyzes the impact of major Indian events — Diwali and the Union Budget — on the NIFTY 50 index using historical price data. It evaluates how the market behaves 15 days before and 15 days after each event from 2006 to 2025, compares average prices, calculates percentage impact, and visualizes long-term trends.
